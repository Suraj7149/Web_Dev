Adobe Notes:- 


Adobe Shortcuts:- 

========================
1. Function Keys
========================
F1              - Help
F2/F3/F4        - Cut / Copy / Paste
F5              - Show/hide Brushes panel
F6              - Show/hide Color panel
F7              - Show/hide Layers panel
F8              - Create new symbol
Ctrl+F8         - Show/hide Info panel
Ctrl+F9         - Show/hide Gradient panel
Ctrl+F10        - Show/hide Stroke panel
Ctrl+F11        - Show/hide Attributes panel
F12             - Revert
Shift+F5        - Show/hide Graphic Styles panel
Shift+F6        - Show/hide Appearance panel
Shift+F7        - Show/hide Align panel
Shift+F8        - Show/hide Transform panel
Shift+Ctrl+F9   - Show/hide Pathfinder panel
Shift+Ctrl+F10  - Show/hide Transparency panel
Shift+Ctrl+F11  - Show/hide Symbols panel
Ctrl+Shift+I    - Show/hide Perspective Grid

========================
3. Viewing Artwork
========================
F                   - Toggle screen modes
Double-click Hand   - Fit imageable area in window
Double-click Zoom   - Magnify 100%
Space               - Switch to Hand tool
Ctrl+Space          - Zoom in
Ctrl+Alt+Space      - Zoom out
Space (while zoom)  - Move Zoom marquee
Ctrl+Alt+Shift+3    - Hide unselected artwork
Alt+Drag Guide      - Convert guide orientation
Ctrl+Shift+Double-click Guide - Release guide
Ctrl+Shift+H        - Show/Hide Artboards
Ctrl+Alt+R          - Show/Hide Artboard Rulers
Ctrl+Alt+0          - View all artboards
Ctrl+Shift+V        - Paste in place
Esc                 - Exit Artboard or Fullscreen mode
Shift+Drag          - Nested artboard
Ctrl+Click          - Select multiple artboards
Ctrl+F6             - Next document
Ctrl+Shift+F6       - Previous document
Ctrl+Alt+F6         - Next document group
Ctrl+Alt+Shift+F6   - Previous document group

========================
4. Drawing
========================
Shift+Drag          - Constrain shape proportions
Spacebar+Drag       - Move shape while drawing
Alt+Drag            - Draw from center of shape
Start+Arrow Keys    - Adjust shape properties (e.g., sides)
Start+C             - Toggle arc open/close
Start+F             - Flip arc
Start+Alt+Drag      - Spiral winds
Start+Ctrl+Drag     - Spiral decay
Start+V/X           - Adjust skew values
Alt+Click Live Trace- Expand Live Trace
[ / ]               - Decrease/Increase Blob Brush size
Shift               - Constrain Blob Brush direction
Shift+D             - Switch drawing modes
Ctrl+J              - Join paths
Shift+Ctrl+Alt+J    - Create corner/smooth join

========================
5. Drawing in Perspective
========================
Shift+P             - Perspective Grid Tool
Shift+V             - Perspective Selection Tool
Ctrl+Shift+I        - Toggle Perspective Grid
5 Key + Drag        - Move perpendicularly
1–4 Keys            - Switch perspective planes
Ctrl+Alt+Drag       - Copy in perspective
Ctrl+D              - Repeat transformation
Shift+D             - Switch drawing modes

========================
6. Keys for Selecting
========================
Ctrl                - Switch to last-used selection tool
Alt                 - Switch to Group Selection
Shift+Click         - Add/subtract from selection
Alt+Click (Magic Wand) - Subtract from selection
Shift/Alt+Drag (Lasso) - Add/subtract with Lasso tool
Caps Lock           - Crosshair for Lasso
Ctrl+Alt+A          - Select in active artboard
Alt+C+O             - Create crop marks
Ctrl+Click x2       - Select behind object

========================
7. Moving Selection
========================
Arrow Keys          - Move selection
Shift+Arrow Keys    - Move 10x
Ctrl+Alt+Shift+2    - Lock deselected
Shift (with tools)  - Constrain to 45°

========================
8. Editing Shapes
========================
Alt (various)       - Switch tools (Pen, Anchor, Scissors, Smooth)
Spacebar+Drag       - Move anchor point (Pen)
Alt+Drag            - Cut straight with Knife
Shift+Alt+Drag      - Cut at 45°/90°
Alt+Shape Mode      - Use Pathfinder shape modes
Alt+Click Region    - Erase Shape Builder region
Shift+Click+Drag    - Marquee Shape Builder

========================
9. Painting Objects
========================
X                   - Toggle fill/stroke
D                   - Default fill/stroke
Shift+X             - Swap fill/stroke
> / <               - Gradient / Color fill mode
/                   - No fill/stroke
Shift+Eyedropper    - Sample gradient color
Alt+Shift+Click     - Append style
Ctrl+/              - Add fill
Ctrl+Alt+/          - Add stroke
Ctrl+Click Gradient - Reset gradient
Alt+O+J             - Mosaic options
[ / ]               - Bristle brush size
1–0                 - Brush opacity

========================
10. Live Paint Groups
========================
Alt+Click (Bucket)          - Sample fill/stroke
Alt+Shift+Click (Bucket)    - Sample image color
Shift (with tool)           - Toggle Bucket options
Double-Click (Bucket)       - Fill across edges
Triple-Click (Bucket)       - Fill all matching faces/strokes
(Also applies to Selection tool with similar shortcuts)

========================
11. Transforming Objects
========================
Alt+Click           - Set origin point
Alt+Drag            - Duplicate and transform
~+Drag              - Transform pattern only

========================
12. Variable Width Points
========================
Shift+Click         - Select multiple width points
Alt+Drag            - Create non-uniform widths
Alt+Drag (Point)    - Duplicate width point
Shift+Drag          - Move multiple points
Delete              - Delete width point
Esc                 - Deselect point

========================
13. Working with Type
========================
Arrow Keys          - Move character/line
Ctrl+Arrows         - Move word/paragraph
Shift+Ctrl+Arrow    - Select word/paragraph
Ctrl+Shift+L/R/C    - Align paragraph
Ctrl+Shift+J        - Justify
Shift+Enter         - Soft return
Ctrl+Alt+K          - Highlight kerning
Ctrl+Shift+X        - Reset scale
Ctrl+Shift+<>       - Adjust font size
Alt+Arrow           - Adjust leading
Double-Click Leading Icon - Set to type size
Ctrl+Alt+Q          - Reset tracking/kerning
Alt+Arrow Keys      - Adjust tracking/kerning
Ctrl+Alt+Arrow Keys - Adjust faster
Alt+Shift+Arrow     - Adjust baseline shift
Shift               - Toggle type modes
Alt                 - Switch Area/Path types

========================
14. Panel Shortcuts
========================
[Actions Panel]
Alt+Click Triangle      - Expand all
Double-Click Folder     - Action Set options
Ctrl+Click Play         - Play one command
Alt+Click New Action    - Start recording

[Brushes Panel]
Double-Click Brush      - Options dialog
Drag to New Button      - Duplicate

[Character & Paragraph Panels]
Arrow Keys              - Adjust small
Shift+Arrow             - Adjust large
Ctrl+Alt+Shift+F        - Font name field

[Color Panel]
Ctrl+Click Bar          - Complement color
Alt+Click Bar           - Nonactive color
Ctrl+Alt+Click Bar      - Nonactive complement
Ctrl+Shift+Click        - Invert fill/stroke
Shift+Click             - Color mode
Shift+Drag              - Sync sliders

[Layers Panel]
Alt+Click Layer Name    - Select all on layer
Alt+Click Eye Icon      - Hide others
Ctrl+Click Eye          - Outline/Preview
Alt+Click Lock Icon     - Lock others
Alt+Click Triangle      - Expand all
[...and more as above]

[Swatches Panel]
Ctrl+Click New Swatch   - Spot color
Ctrl+Shift+Click        - Global color
Alt+Drag                - Replace swatch

[Transform Panel]
Shift+Enter             - Apply, keep focus
Alt+Enter               - Apply, copy object
Ctrl+Enter              - Scale proportionally

[Transparency Panel]
Alt+Click Mask          - Edit grayscale
Shift+Click Mask        - Disable mask
Click + Arrows          - Adjust opacity
